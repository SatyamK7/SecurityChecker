# 🔍 URL Checker - Browser Extension

A simple browser extension that checks and validates URLs visited by the user. It can be used to monitor URLs, match patterns, or block untrusted domains.

## 🚀 Features

- Automatically checks the URL of each visited page.
- Matches URLs against a custom pattern or list.

## 🧩 Supported Browsers

- Chrome
- Edge
- Firefox (with minor adjustments)
